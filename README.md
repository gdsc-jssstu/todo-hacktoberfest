# todo
In the root Folder:
To install the dependencies: ```npm install``` 

To connect to mongodb: 

```mongod```

```mongo```

```show dbs``` (in case you want to see the list available)

```use todo_expresss```

To get the list of all tasks in cml:
```db.todos.find()```

To delete all the tasks:
```db.todos.deleteMany({})```
